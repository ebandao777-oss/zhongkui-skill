# 钟馗.Skill 快速上手

> 直来直去，三步上手。

## 安装

把 `zhongkui-skill/` 目录放入 skills 目录即可。

## 基本用法

### 全量审查

```
审 E:\skills\some-skill
```

执行完整三层审查，输出结构化裁定报告。

### 快速审计（秒级）

```
快审 E:\skills\some-skill
```

仅 Layer 1 静态审计，适合批量初筛。

### 只查依赖

```
审依赖 E:\skills\some-skill
```

仅 Layer 3 供应链溯源。

### 只查行为

```
审行为 E:\skills\some-skill
```

仅 Layer 2 行为模拟，适合深度复查。

## 输出解读

| 裁定 | 含义 | 行动 |
|:---|:---|:---|
| ✅ 干净 | ≥ 85 分 | 放心装 |
| ⚠️ 可疑 | 60-84 分 | 看一眼扣分项再决定 |
| 🚫 恶意 | < 60 或命中否决 | 别装，上报 |

## 常见问题

**Q: 为什么我的 Skill 被判 R8？**
A: 八成是 SKILL.md 里有零宽字符。用 `审` 命令会告诉你具体行号。

**Q: 审查多久？**
A: Layer 1 秒级。Layer 2 约 30 秒（12 场景模拟）。Layer 3 取决于依赖数量。

**Q: 误报怎么办？**
A: 记下扣分项的具体内容，反馈给钟馗。多引擎仲裁机制保证不会单一规则误杀。
